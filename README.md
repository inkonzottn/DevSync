# 🚀 IT Project Management System

Корпоративна система для управління ІТ-проєктами, командами розробників та фінансовою звітністю.

## 🛠 Технологічний стек
* **Мова:** Java 17+
* **Фреймворк:** Spring Boot (Web, Data JPA)
* **Безпека:** Spring Security (RBAC: Admin, Manager, Developer, Customer)
* **Шаблонізатор:** FreeMarker
* **Збірка:** Maven

## 🔑 Основний функціонал (Features)
* Розмежування прав доступу на основі ролей.
* **Admin Panel:** Повне управління системою, моніторинг усіх процесів.
* **Manager Workspace:** Створення та редагування проєктів, призначення розробників (`Developers`), контроль завдань (`Assignments`), виставлення рахунків (`Invoices`).
* Кешування частих запитів до бази даних для оптимізації Database Hits.

## ⚙️ Інструкція із запуску (Getting Started)

1. **Клонуйте репозиторій:**
   `git clone <URL_вашого_репозиторію>`
2. **Зберіть проєкт за допомогою Maven:**
   `mvn clean package`
3. **Запустіть застосунок:**
   `java -jar target/oop-np-0.0.1-SNAPSHOT.jar`
